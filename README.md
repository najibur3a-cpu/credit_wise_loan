# 💳 Credit Wise Loan Prediction

A **machine learning project** that predicts **loan approval status** using applicant financial and demographic information.
This project demonstrates a full **data science workflow**, including **data preprocessing, exploratory data analysis (EDA), model training, and evaluation**.

---

# 📌 Project Overview

Financial institutions receive many loan applications every day.
This project uses **machine learning algorithms** to analyze applicant data and determine whether a loan application is likely to be **approved or rejected**.

The goal is to build a model that helps financial institutions make **data-driven lending decisions**.

---

# 🧠 Machine Learning Workflow

The project follows a standard **ML pipeline**:

1️⃣ Data Collection
2️⃣ Data Cleaning & Preprocessing
3️⃣ Exploratory Data Analysis (EDA)
4️⃣ Feature Engineering
5️⃣ Model Training
6️⃣ Model Evaluation

---

# 📊 Dataset

The project uses a dataset:

```
loan_approval_data.csv
```

The dataset contains features such as:

* Applicant Income
* Loan Amount
* Loan Amount Term
* Credit History
* Gender
* Marital Status
* Education
* Self Employment
* Property Area

Target Variable:

```
Loan_Status
```

Where:

* **Y → Loan Approved**
* **N → Loan Rejected**

# 🤖 Machine Learning Models Used

The project compares multiple classification algorithms:

### Logistic Regression

Used as a baseline model for loan approval prediction.

### Naive Bayes

A probabilistic model useful for classification tasks.

### K-Nearest Neighbors (KNN)

A distance-based model that classifies data based on nearest neighbors.

# 📈 Model Evaluation

The models are evaluated using:

* Accuracy

These metrics help determine how well the model predicts loan approval outcomes.
